🌟 これは何？
安くて早くて簡潔。保険営業に特化したAIボイスレコーダーです。

商談を録音するだけで、提出フォーマットに準拠した商談記録が自動生成されます。

✨ できること
録音するだけで商談記録が完成 — お客様の同意を確認して録音、停止するだけ。文字起こしから報告書の作成までを自動で行います。
個人商談・法人商談を自動判別 — AIが会話内容から判別し、それぞれの様式で出力します。テンプレートを選ぶ操作は不要です。
保険用語に強い多層辞書 — 保険・金融・税務・疾病の専門用語を収録。「解約返戻金」「特定三疾病」「上皮内がん」といった、汎用の文字起こしでは崩れやすい用語を正しく記録します。
提案した数字をそのまま記録 — 給付日額、保険料、払込期間などの数値を、話した通りに書き起こします。聞き取りが不確かな数値には「(要確認)」を付けます。
タップひとつでメール送信 — 作成した記録を、登録済みのメールアドレスへその場で送れます。
長時間の録音に対応 — 長時間の商談も自動で分割処理します。通話などで中断しても、録音データを失いません。

📱 使い方
月額課金はゼロ

かかるのはAI処理の実費のみです。2時間の商談でも数十円程度。アプリ内で1件ごとの概算コストと今月の合計を確認できます。

履歴は端末内に保存

録音と商談記録は、お使いの端末のブラウザ内にのみ保存されます。外部に送信されるのは文字起こしと要約の処理分のみで、会話データをクラウドに保管する仕組みは持たせていません。

アカウント登録は不要

インストールもログインも不要です。ブラウザで開いて、ご自身のAPIキーを設定すればすぐに使えます。

設計上の工夫

多層構造の用語辞書 文字起こしの誤変換を、段階的に補正する仕組みを持っています。誤変換パターンの機械置換に加え、整形AIに用語集と文脈判定ルールを渡すことで、「保障/補償/保証」のように音が同じで文脈によって使い分ける語も判別します。実機テストで発見した誤変換を辞書へ追加していく運用です。

長時間録音と中断への対応 録音を一定時間ごとに分割し、順次処理することで長時間の商談に対応しています。分割は同時に中断対策も兼ねており、通話やアプリ切り替えが発生しても、その時点までの録音を確保します。送信に失敗した録音は端末内に保持され、通信復帰後に再送できます。

捏造を防ぐプロンプト設計 商談記録は事実の正確さが最優先されるため、録音に存在しない情報を補完しない設計にしています。情報が不足する項目は無理に埋めず「(記載なし)」とし、担当者が説明に用いた例え話を顧客の関心として記録しないなど、実運用のテストで見つかった誤りへの対策を積み上げています。

開発について

TAKOYAKI PROJECT による個人開発です。保険営業の事務作業を間近で見る中で、「商談後の記録作成に時間がかかる」「専門用語が文字起こしで崩れる」「顧客情報を外部に預けたくない」という課題から制作しました。

汎用の議事録ツールとの違いは、保険営業という特定の業務に絞り込んでいる点にあります。提出様式に合わせた出力、業界用語への対応、端末内保存という設計は、いずれもこの用途に特化したものです。
